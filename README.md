# Задания по курсу

## Настройка
Перед началом использования проекта, нужно выполнить установку зависимостей

```shell
export YC_TOKEN=$(yc iam create-token)
export YC_CLOUD_ID=$(yc config get cloud-id)
export YC_FOLDER_ID=$(yc config get folder-id)

npm i
```
